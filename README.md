# project-aether

Notes, code examples, and best practices for learning and developing with Vulkan.

## Development Environment Setup

To set up the development environment and install all required dependencies, run:

```
just install
```


This will execute the bootstrap script located at `tools/dev/bootstrap.ps1` (on Windows) or the corresponding shell script if available.

See [docs/RESSOURCES.md](docs/RESSOURCES.md) for a curated list of useful resources and links.
